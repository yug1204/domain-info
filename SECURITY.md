We take the security of DomainRecon seriously. If you believe you have found a security vulnerability, please do **not** open a public issue. Instead, please report it through one of the following methods:

1. **Email:** [INSERT EMAIL ADDRESS]
2. **Private Message:** [INSERT OTHER METHOD IF APPLICABLE]

Please provide a detailed description of the vulnerability, including:
* Steps to reproduce the issue

## Ethics and Usage

As stated in the README, this tool is for **educational and ethical security research purposes only**. We do not condone its use for unauthorized reconnaissance or any illegal activities. Users are responsible for ensuring they have permission to scan targets.
